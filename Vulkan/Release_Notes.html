Vulkan SDK for Linux Release Notes

Vulkan SDK 1.1.108.0 Release Notes

Version 1.1.108.0 for Linux
June 14, 2019

This SDK supports Vulkan API revision 1.1.108. The prior SDK supported Vulkan API revision 1.1.106.

Please refer to the Getting Started Guide for a list of supported Linux distributions.

Drivers: It is recommended that you update Vulkan-capable hardware drivers to the latest recommended version from the hardware vendor. A list of Known Issues appears at the end of this document with more information.

Overview of new features in SDK 1.1.108.0

  • GPU-assisted validation code was updated to include validation for scenarios that arise when the VK_EXT_descriptor_indexing extension is enabled. Refer to this document to enable GPU assisted validation. For full details of GPU Assisted Validation refer to this white paper.
  • The Linux Tar SDK now provides one script that will clone and build repositories useful for Vulkan development. Pre-built binaries are still included in the Linux Tar SDK but if there are any compatibility issues with the system, this new script can be used to clone and build them. See the Linux Getting Started guide for more information.
  • Since the release of SDK 1.1.106.0, the following extensions have been added to the Vulkan spec and may be supported by a Vulkan device. To query extensions available to a given physical device, call vkEnumerateDeviceExtensionProperties() or contact the hardware vendor.
VK_KHR_uniform_buffer_standard_layout
VK_NV_framebuffer_mixed_samples_coverage_reduction_mode
VVK_EXT_headless_surface
  • Bug fixes (see the SDK Components section below)
  • Improved validation coverage. VUID validation coverage documentation (validation_error_database.csv and an HTML version) has been updated to represent the validation layer VUID coverage status as of this 1.1.108.0 SDK. validation_error_database.csv can be found locally in the SDK directory once the SDK is installed on the system and an HTML version of this information can be found here.
